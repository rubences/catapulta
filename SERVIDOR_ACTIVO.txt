   ✓ Panel de construcción interactivo
   ✓ Visualización en tiempo real del estado
   ✓ Campo de batalla con canvas HTML5
   ✓ Barras de progreso animadas
   ✓ Sistema de selección de enemigos
   ✓ Efectos visuales y animaciones
   ✓ Responsive design

📋 FLUJO DEL JUEGO:

   1. Crear Catapulta → Darle un nombre
   2. Agregar Materiales → Usar los selectores
   3. Construir → Ver la catapulta aparecer
   4. Generar Oleada → Enemigos aparecen
   5. Seleccionar Enemigo → Click en la lista
   6. Disparar → Observar resultados
   7. Reparar/Mejorar → Según sea necesario

💡 VENTAJAS DE LA VERSIÓN WEB:

   • Funciona en cualquier navegador
   • No requiere display gráfico (X11)
   • Compatible con dev containers
   • Interfaz moderna y responsive
   • Visualización en tiempo real
   • Animaciones fluidas

🛠️  PARA DETENER EL SERVIDOR:
